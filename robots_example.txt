Installation Steps
1. MySQL Database Setup
Install MySQL（Recommended MySQL version: 5.7）


docker pull mysql:5.7
Run MySQL 5.7 container in detached mode, map port 3306, persist data to host, set root (123456), create 'zencart' database, and configure UTF-8 character set



mkdir -p ~/docker/mysql57/data
docker run -d --name mysql57   -p 3306:3306  -v ~/docker/mysql57/data:/var/lib/mysql   -e MYSQL_ROOT_PASSWORD=123456  -e MYSQL_DATABASE=zencart  mysql:5.7 --character-set-server=utf8 --collation-server=utf8_general_ci
docker ps -a  
You can see that the MySQL container has started.

image-20250718-073311.png
2. PHP 7.4 Installation
Since Homebrew no longer maintains PHP 7.4, we'll use a third-party repository:



brew tap shivammathur/php
brew install shivammathur/php/php@7.4
brew link --overwrite --force php@7.4
php -v
3. Zen Cart Installation Process
1.Enter the ZenCart project and run the command below to start it

php -S localhost:8888 -t .

2.Navigate to http://localhost:8888/zc_install/index.php

3.Follow the on-screen installation wizard, configure the Database host (127.0.0.1), database username and password with the root credentials set earlier(root ,123456,zencart).

Tip: Check the box next to "Load demo data into this database?"; this will generate test product data.

4.Remove the zc_install directory as prompted after successful installation
